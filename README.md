# tiny-budget

A tiny CLI budget tracker for fast daily money check-ins.

## Features
- Add income and expense transactions
- View monthly budget summary
- List recent transactions
- Local JSON persistence (no cloud required)

## Stack
- Node.js 20+
- TypeScript
- tsx for local execution/tests

## Setup
```bash
npm install
```

## Run
```bash
npm run dev
```

## Test
```bash
npm test
```

## Docker
```bash
docker build -t tiny-budget .
docker run --rm tiny-budget
```
