# tiny-budget — MVP Spec

## App Description
Tiny Budget is a minimal command-line budgeting app that helps users quickly record income/expenses and view a simple remaining-balance snapshot without complex setup.

## Target User
- Solo builders, students, or freelancers who want a lightweight personal budget tracker.
- Users who prefer terminal tools and fast, low-friction workflows.

## MVP Scope
1. Add transactions from CLI with amount, category, and optional note.
2. Persist entries in a local JSON file.
3. Show monthly summary: total income, total expenses, net balance.
4. List recent transactions.
5. Basic input validation and clear error messages.

## Non-Goals (for MVP)
- Multi-user auth
- Cloud sync
- Bank integrations
- Complex analytics dashboards
